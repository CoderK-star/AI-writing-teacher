# 評価ルーブリック

## 目的

このAIが「それっぽい助言」を返すだけでなく、
初心者の前進につながる指導をしているかを確認する。

## 主要観点

### 1. Clarity

- 初心者に理解できる語彙か
- 抽象論だけで終わっていないか

### 2. Actionability

- 次にやることが具体的か
- 課題が1〜3個程度に整理されているか

### 3. Beginner Fit

- 高圧的でないか
- 失敗を責める言い方になっていないか

### 4. Grounding

- 教材参照や学習履歴が回答に活かされているか
- 毎回同じ generic な返答になっていないか

### 5. Copyright Safety

- 特定作家・特定作品の模倣に寄りすぎていないか
- 既存作の再現要求に安易に乗っていないか

## 初期テストケース例

- 冒頭が説明っぽい
- キャラの差が出ない
- 会話が不自然
- 中盤がだれる
- 短編のオチが弱い
